## 1. 品牌标志 Logo

<img src="https://raw.githubusercontent.com/hannahhoo47/Etna-Asset/main/logos/png/etna-horizontal-red.png" width="440" alt="ETNALabs">

### 1.1 标志构成

标志由字标 ETNA、副标 Labs、以及可选 tagline 构成。字母 **A** 被替换为埃特纳火山（Mount Etna）的山形：实心红色三角（山体）+ 顶部一道白色缺口（火山口负形，同时充当 A 的横杠）。这是整套识别系统里唯一的图形元素，也是标志不可替换的核心——**任何情况下不得用普通字母 A 代替。**

### 1.2 三种标准组合（Lockup）

| 组合 | 说明 | 使用场景 |
|---|---|---|
| **竖版** | ETNA 上 / Labs 下 | 头像、印章、方形版位 |
| **横版** | ETNALabs 单行连写 | 网站页眉、邮件签名、文档页脚 |
| **横版 + tagline** | ETNALabs / Deep Conviction. Long View. | 名片、PPT 首页、封面 |

> **不含"字标 + tagline"（ETNA + tagline、不带 Labs）版本**——该版本不作为标准组合使用。

## 2. 安全距离 Clear Space **[源文件]**

标志四周必须留出 **X** 的空白，X 内不得出现任何文字、图形、图片边缘或版心线。

```
        ┌─────── X ───────┐
        │  ┌───────────┐  │
        X  │   LOGO    │  X        X = 1/2 × lockup 高度
        │  └───────────┘  │
        └─────── X ───────┘
```

**X = 标志（含 tagline，如有）总高度的一半。** 从矢量文件实测，各标准组合该比值均为 0.50（例如 91/183.55、71.08/143.36、65/129.78）。

计算示例：横版标志放置高度为 40px 时，四周留白 ≥ 20px。

## 3. 最小尺寸 Minimum Size

- **[源文件]** 印刷：标志高度不小于 **6mm**。
- **[建议]** 屏幕端最小高度 **24px**（6mm ≈ 22.7px @96dpi）。低于此尺寸时，火山口的实心块会与 A 的山体轮廓糊在一起，失去辨识度。

## 4. 色彩 Color **[源文件]**

视觉规范 PDF（[`source/ETNA-Visual-Identity-Guidelines.pdf`](https://github.com/hannahhoo47/Etna-Asset/blob/main/source/ETNA-Visual-Identity-Guidelines.pdf)）的色彩页明确定义了三色，含 HEX 与 CMYK：

| 色 | HEX | RGB | CMYK |
|---|---|---|---|
| **Etna Red** | `#C43426` | 196, 52, 38 | **C29 M92 Y95 K0** |
| **Black** | `#000000` | 0, 0, 0 | C0 M0 Y0 K100 |
| **White** | `#FFFFFF` | 255, 255, 255 | C0 M0 Y0 K0 |

> ℹ️ 源文件红色行的标签印作 **"CMKY: 29 92 95 0"**——`CMKY` 是 `CMYK` 的笔误，数值按标准 CMYK 顺序即 **C29 M92 Y95 K0**。
> HEX 用于屏幕，CMYK 用于印刷。不同 ICC profile 与纸张仍会有细微差异，首次上机建议打样确认观感。

### 4.1 中性色阶 **[建议]**

源文件未定义中性色。以下为网页与文档系统的补充建议，基于 Etna Red 的暖调偏移：

| Token | HEX | 用途 |
|---|---|---|
| `--ink` | `#191715` | 正文 |
| `--muted` | `#6D6860` | 次要文字 |
| `--line` | `#D8CEC2` | 分隔线、描边 |
| `--bg` | `#F4F0EA` | 页面底色 |
| `--paper` | `#FFFDF9` | 卡片、浮层 |

### 4.2 ⚠️ 现有网站实现与规范不符

`Etna Website/app/globals.css` 当前 `--brand: #c45a48`，比品牌红 `#C43426` 更浅更粉。网站、邮件签名、名片工具都在用这个值，**需统一改为 `#C43426`**，或由设计明确 `#c45a48` 是否是有意的网页浅色版本。

```css
:root {
  --brand:      #C43426;  /* Etna Red — 品牌主色 */
  --brand-dark: #8F2620;  /* [建议] hover / pressed */
  --brand-tint: #FBEDEB;  /* [建议] 浅底、选中态 */
}
```

| **阿里妈妈方圆体 VF** | [`fonts/AlimamaFangYuanTiVF-Thin.ttf`](https://github.com/hannahhoo47/Etna-Asset/blob/main/fonts/AlimamaFangYuanTiVF-Thin.ttf) | 中文 | 免费商用 |
| **Avenir Next** | —（未入库） | 西文正文 / UI | ⚠️ Apple 商业字体，因授权原因不随仓库分发 |

> 设计稿原始文件名 `finSans-Regular-7 2.ttf` 实为 **Josefin Sans**（内部 name table 证实），本仓库已更名为 `JosefinSans-Regular.ttf`。

### 层级 **[建议]**

| 层级 | 字体 | 说明 |
|---|---|---|
| **Logotype** | 定制字形 | ETNA 四字母（含火山 A）为定制绘制，**不可用任何字体重排**，只能调用矢量文件 |
| **Display / tagline** | Josefin Sans | 几何无衬线，低 x-height，与标志同源。仅用于大字号 |
| **西文正文 / UI** | Avenir Next Regular / Medium | 长文可读性优于 Josefin Sans |
| **中文** | 阿里妈妈方圆体 VF | 圆润几何，与 Josefin Sans 调性匹配 |

Josefin Sans 只适合大字号；小于 16px 的正文一律用 Avenir Next。网页端若无 Avenir Next webfont 授权，用 Inter 替代。

## 7. 禁用规范 Don'ts **[建议]**

- ❌ 不得拉伸、压缩、倾斜标志——等比缩放
- ❌ 不得替换标志颜色（红 / 黑 / 白三色以外）
- ❌ 不得为标志添加描边、阴影、渐变、外发光
- ❌ 不得旋转标志
- ❌ 不得重排 lockup 内部元素的相对位置与大小
- ❌ 不得把火山 A 换成普通字母 A
- ❌ 不得在花色照片或低对比背景上直接放标志——加纯色底或半透明遮罩
- ❌ 不得侵入 clear space

### 背景适配

| 背景 | 使用色版 |
|---|---|
| 白 / 浅色底 | 红版（首选）或黑版 |
| 黑 / 深色底 | 白版 |
| Etna Red 底 | 白版 |
| 照片 | 白版 + 深色遮罩（不透明度 ≥ 40%） |
